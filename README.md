# Task Management SaaS

## Live Demo
Frontend: https://your-vercel-link  
Backend: https://your-render-link  

---

## Features
- JWT Authentication
- User-specific tasks
- Create / Update / Delete tasks
- Protected routes
- Clean UI (React + Tailwind)

---

## Tech Stack
- React + Vite
- Node.js + Express
- PostgreSQL + Sequelize
- Tailwind CSS

---

## Setup

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev
